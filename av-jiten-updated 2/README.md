# AV辞典

GitHub Pages にそのまま置ける、APIなし版の個人用コレクション管理Webアプリです。

## 含まれる機能

- 18歳以上確認
- 品番検索フォーム
- 女優名検索フォーム
- FANZA / Google / X の外部検索ボタン
- 手入力登録
- お気に入り、視聴済み、メモ
- サムネURL登録と固定サイズ表示
- ローカル保存
- JSON書き出し・読み込み
- プライバシーポリシー表示
- PWA対応

## 公開方法

このフォルダ内のファイルを GitHub リポジトリにアップロードし、Settings → Pages から公開してください。

データは各端末のブラウザ内に保存されます。別端末に移したい場合は「書き出し」でJSONを保存し、別端末側で「読み込み」を使ってください。
